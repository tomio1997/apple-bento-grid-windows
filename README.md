## What is apple-bento-grid?

apple-bento-grid is a presentation card generator centered on an Apple-like bento grid layout. It is built to output self-contained HTML, so you can go from raw content to a finished visual card without depending on external design applications.

This repository is intended for AI-driven coding flows, including agent-based environments like Claude Code and Codex. It is particularly handy when you need consistent, layout-conscious cards for presentations, dashboards, or data-visualization screens on Windows.

Example settings pattern:
